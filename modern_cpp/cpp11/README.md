# C++11 特性

常用特性练习：

- auto / decltype
- lambda 表达式
- 右值引用 / 移动语义
- 智能指针 (unique_ptr, shared_ptr, weak_ptr)
- 范围 for 循环
- initializer_list
- nullptr
- constexpr
- 线程库 (std::thread, std::mutex)
- 原子变量 (std::atomic)
- 可嵌套锁 (std::recursive_mutex)
- 读写锁 (std::shared_timed_mutex + std::shared_lock / std::unique_lock)
- std::function / std::bind
- 变参模板
- enum class
