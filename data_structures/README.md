# Data Structures

Practice implementations of common data structures in C++, from basics to advanced.

## Linear Structures
```
├── Array
├── Vector
├── List (Linked List)
├── Stack
├── Queue
└── Deque (Double-ended Queue)
```

## Tree Structures
```
├── Binary Tree
├── BST (Binary Search Tree)
├── AVL Tree
├── Red-Black Tree
├── B-Tree / B+ Tree
├── Trie (Prefix Tree)
├── Segment Tree
├── Binary Indexed Tree (Fenwick Tree)
└── Heap
```

## Hash Structures
```
├── HashMap
├── HashSet
├── Bloom Filter
└── Consistent Hash
```

## Graph Structures
```
├── Adjacency Matrix
├── Adjacency List
├── Union-Find (Disjoint Set)
└── Topological Sort
```

## Advanced Structures
```
├── Skip List
├── Ring Buffer
├── Lock-Free Queue
├── LRU Cache
└── Time Wheel
```
